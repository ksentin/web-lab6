Логічні оператори

  • not: Використовується для вибору стилів, які застосовуються до елементів, якщо умова не виконується.
    
        @media not (max-width: 1900px) {
            /* Стилі, які будуть застосовуватися до елементів, коли ширина екрану не більше 1900px */
            .content {
                width: 15%;
            }
        }
  • and: Використовується для вибору стилів, які застосовуються до елементів, якщо одночасно виконуються дві або більше умов.
    
        @media only screen and (max-width: 750px) {
            /* Стилі, які будуть застосовуватися лише до елементів на екранах з максимальною шириною 750px */
            .content {
                width: 40%;
            }
        }
  • only: Використовується для того, щоб стилі застосовувалися лише до певного типу пристрою та умови.
    
        @media only screen and (max-width: 750px) {
            /*стилі для .content будуть застосовані тільки до екранів з максимальною шириною 750px*/
            .content {
                width: 40%;
            }
        }
