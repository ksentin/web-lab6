Основні характеристики цієї верстки включають:

  • Властивість display: flex; встановлює контейнер .task як flex-контейнер, що дозволяє гнучко розташовувати його дочірні елементи.
  • Відсоткові розміри: Ширина та висота контейнера .task задаються у відсотках (height: 100vh;), що дозволяє контейнеру займати весь видимий висоту екрану.
  • Гнучкість розміщення елементів за допомогою flex: Дочірні елементи контейнера .task, такі як .header, .footer та .content, використовують flex-властивості для розміщення. Наприклад, .content має flex: 1;, що дозволяє йому займати усі доступні залишкові простори у контейнері.
  • Гнучкість ширини блоків: Для блоків .column використовуються відсоткові значення ширини (наприклад, flex: 1;, flex: 2;), що дозволяє їм гнучко розміщуватися відносно один одного та змінювати свою ширину в залежності від розміру контейнера.

        <style>
        .task {
            font-family: Arial, sans-serif;
            margin-left: 20px;
            margin-right: 20px;
            display: flex;
            flex-direction: column;
            height: 100vh;
        }
        
        .header, .footer {
            background-color: blue;
            color: #fff;
            padding: 20px;
            text-align: center;
            flex-shrink: 0;
            border: 1px solid #000000;
        }
        
        .content {
            overflow: hidden;
            display: flex;
            flex: 1;
        }
        
        .column {
            flex: 1;
            display: flex;
            border: 1px solid #000000;
            text-align: center;
        }

        .column h3 {
            margin: 0; 
            text-align: center; 
        }

        .column:nth-child(2) {
            flex: 2;
            background-color: #f2f2f2;
        }

        .column:nth-child(1),
        .column:nth-child(3) {
            flex: 1;
            background-color: yellow;
        }

        .clearfix::after {
            content: "";
            display: table;
            clear: both;
        }
        </style>
