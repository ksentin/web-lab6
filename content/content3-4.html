Фіксована таблична верстка

1
2 3 4 5
6

Основні характеристики цієї верстки включають:

  • Встановлюються конкретні значення ширини і висоти таблиці, що залишаються сталі незалежно від розміру вмісту.
  • Встановлюється параметр table-layout: fixed;, щоб ширина стовпців була фіксованою і не змінювалася в залежності від вмісту.
  • Використання CSS для стилізації: Застосовуються стилі CSS для кожного елемента таблиці, таких як td і tr, для відображення кольорів фону, рамок тощо.

        <style>
            table {
                width: auto;
                height: 80vh;
                table-layout: fixed;
                border-collapse: separate;
                border-spacing: 10px;
            }
        
            tr:nth-child(1),
            tr:nth-child(3) {
                background-color: yellow;
            }
            tr:nth-child(2) td:nth-child(3){
                background-color: white;
            }
            tr:nth-child(2) td:nth-child(1) {
                background-color: blue;
            }
            tr:nth-child(2) td:nth-child(2),
            tr:nth-child(2) td:nth-child(4){
                background-color: red;
            }
            td {
                border: 1px solid black;
                line-height: 20px;
            }
            tr:nth-child(1) td[colspan="4"],
            tr:nth-child(3) td[colspan="4"] {
                line-height: 20px;
                height: 20px;
            }
            tr:nth-child(2) td {
                line-height: 150px;
                height: 150px;
            }
            .white-box {
                background-color: white;
                margin-left: 10px;
                width: 300px; 
                height: 20px;
                display: inline-block;
                border: 1px solid black;
            }
            .numb {
                padding: 10px;
                text-align: right;
            }
        </style>
        

            <table>
                <tr>
                    <td colspan="4"><span class="white-box"></span><span class="numb">1</span></td>
                </tr>
                <tr>
                    <td>2</td>
                    <td>3</td>
                    <td>4</td>
                    <td>5</td>
                </tr>
                <tr>
                    <td colspan="4">6</td>
                </tr>
            </table>
