Основні характеристики цієї верстки включають:

  • Клас .column використовує властивість float: left;, щоб розмістити блоки один поруч з одним горизонтально. Це призводить до того, що блоки .column розташовуються один поруч з одним зліва направо.
  • Кожен блок .column має задану фіксовану ширину (width) відсотково або в пікселях. Це призводить до того, що ширина кожного блоку фіксована і не змінюється в залежності від розміру екрану.
  • Властивість height також задається для кожного блоку .column, що призводить до того, що висота кожного блоку фіксована і не змінюється в залежності від контенту або розміру екрану.

        <style>
        .task {
            font-family: Arial, sans-serif;
            margin-left: 20px;
            margin-right: 20px;
        }
        
        .header, .footer {
            background-color: blue;
            color: #fff;
            padding: 20px;
            text-align: center;
            flex-shrink: 0;
            border: 1px solid #000000;
        }
        
        .content {
            overflow: hidden;
        }
        
        .column {
            float: left;
            border: 1px solid #000000;
            height: 310px;
            text-align: center;
        }

        .column:nth-child(2) {
            width: 49%;
            background-color: #f2f2f2;
        }

        .column:nth-child(1),
        .column:nth-child(3) {
            width: 25%; 
            background-color: yellow;
        }

        .clearfix::after {
            content: "";
            display: table;
            clear: both;
        }
        </style>
