Селектори в CSS

Селектори в CSS - це патерни, за допомогою яких визначається, які елементи HTML-документа повинні стилізуватися. Селектори вибирають HTML-елементи на основі їх ідентифікаторів, класів, типу, атрибутів та інших характеристик.

  • CSS селектор id:

Селектор id використовує атрибут id HTML елемента для вибору визначеного елемента. Ідентифікатор (id) елементу є унікальним на сторінці, тому селектор id використовується для вибору одного унікального елемента.

Щоб обрати елемент з визначеним ідентифікатором, потрібно написати символ хешу (#), а потім id елемента.

  • CSS селектор class:

Селектор class обирає HTML елементи з визначеним атрибутом класу. Щоб обрати елементи з визначеним класом, потрібно ввести символ крапки (.), а потім ім’я класу.

  • Універсальний CSS селектор:

Універсальний селектор (*) виділяє всі елементи HTML на сторінці.

  • CSS Селектор групування:

Селектор групування обирає всі HTML елементи з однаковими визначеннями стилю.
