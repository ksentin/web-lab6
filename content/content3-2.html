У проєкті використовується гібридний макет, який комбінує різні техніки верстки, такі як флексбокс (Flexbox) і сітка CSS (Grid).

  • Flexbox використовується для організації головної частини сторінки для розміщення елементів у різних напрямках і вирівнювання їх.
  • CSS Grid використовується для організації макету сторінки у вигляді сітки, де різні елементи можуть займати різну кількість рядків і стовпців.
