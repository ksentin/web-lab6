Висновки:

У ході виконання лабораторної роботу мною були придбати практичні навички роботи з масивами у js-сценаріях та організовано реалізацію програм засобами мови JAVASCRIPT.

У даному завданні були реалізовані три основні сценарії на JavaScript, кожен з яких вирішує певну задачу:

  • Обчислення суми елементів масиву між максимальним та мінімальним значеннями: Для цього було написано функцію sumBetweenMinMax, яка обчислює суму елементів масиву, розташованих між максимальним і мінімальним значеннями. Сортування масиву за допомогою швидкого сортування: Була створена функція quickSort, яка сортує масив за допомогою алгоритму швидкого сортування.
  • Розробка програми-будильника, яка мала функції установки та вимкнення будильника, введення дати та часу, а також видачу повідомлення при досягненні встановленого часу. Для цього були створені функції setAlarm, disableAlarm і updateTime.
  • Створення сторінок входу та реєстрації на власному веб-сайті: Було реалізовано сторінки входу (signin.html) та реєстрації (registration.html). На цих сторінках застосовано JavaScript для перевірки правильності введених даних користувачем перед відправленням форми. Для цього використовуються функції isValidName для перевірки імені та прізвища, isValidEmail для перевірки електронної пошти, а також функції для відображення повідомлень про помилки і очищення попередніх повідомлень.

Усі три завдання були успішно виконані, що показує вміння використовувати JavaScript для створення функціональності на веб-сайті та для реалізації корисних інструментів для користувача.
